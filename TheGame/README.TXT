PROJECT TITLE: ERROR 101
PURPOSE OF PROJECT: To teach students how to use the number system in programming
VERSION or DATE: v_1
HOW TO START THIS PROJECT: Option Click The testGame class and click on the void main(string[]args) to run the application
The game will start with HomeScreen where the player can enter their names. After the HomeScreen there is a NavigationScreen
where the player selects the stage they wish to participate in
AUTHORS: BT MAKUNYANE and MZ MOLEFE
USER INSTRUCTIONS: The player has to convert a given Integer value to either binary, hexadecimal, or octal number system.
